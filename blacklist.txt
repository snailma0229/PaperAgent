# PaperAgent Blacklist — 禁止访问的域名列表
# 格式：每行一个域名，# 开头为注释，空行忽略
# 匹配规则：完整域名匹配 + 子域名匹配（e.g. github.com 同时拦截 api.github.com）
#
# 用法：
#   /paper-agent "paper.pdf" — mode: blacklist — blacklist-file: "blacklist.txt"
#
# 也可以配合内联参数一起用（会自动合并去重）：
#   /paper-agent "paper.pdf" — mode: blacklist — blacklist: "extra.com" — blacklist-file: "blacklist.txt"

# ---- 示例：禁止访问特定 GitHub 仓库所在账号 ----
# github.com/competitor-lab

# ---- 示例：禁止访问论文评测排行榜 ----
# paperswithcode.com
# sotabench.com

# ---- 示例：禁止访问特定博客或泄露实现 ----
# some-leaked-impl.com
